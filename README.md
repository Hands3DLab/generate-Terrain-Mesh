# Generatting terraain mesh by longitude and latitude

## Description

The code was generated by Gemini3, used to generate terrain mesh by longitude and latitude. You should replace the API_KEY_HERE with your own API key which you can get from https://portal.opentopography.org/user/register.

这段代码是用来生成地形3D地形实体，你需要替换代码中的API_KEY_HERE为你自己的API key。当然这段代码是由Gemini3生成的。
你需要在https://portal.opentopography.org/user/register注册一个账号，获取你的API key。

## Usage

1. Run `pip install -r requirements.txt` to install the required modules, you need pre-install Python3 and pip.
2. Replace the API_KEY_HERE with your own API key.
3. Replate parameters in the code, such as LAT, LON, GEO_WIDTH, MODEL_WIDTH, BASE_HEIGHT, Z_SCALE.
4. Rund `python3 ./generateTerrain.py` to generate the terrain mesh.
5. Import the generated STL file into your 3D Prints software. e.g. Bambustudio or Blender, Shapr3D, etc.

1. 运行`pip install -r requirements.txt`安装所需的模块，你需要先安装Python3和pip。
2. 替换代码中的API_KEY_HERE为你自己的API key。
3. 替换代码中的参数，如LAT, LON, GEO_WIDTH, MODEL_WIDTH, BASE_HEIGHT, Z_SCALE。
4. 运行`python3 ./generateTerrain.py`生成3D地形文件。
5. 将生成的STL文件导入到你的3D打印软件中。如Bambustudio或Blender, Shapr3D等。
